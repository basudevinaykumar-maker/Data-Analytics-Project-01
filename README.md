# Data-Analytics-Project-01

## 📌 Project Overview

This project focuses on analyzing a business dataset using Microsoft Excel. The goal is to clean the data, perform data analysis, and create an interactive dashboard that provides meaningful business insights.

## 🎯 Objectives

* Clean and preprocess raw data.
* Perform data analysis using Excel.
* Create Pivot Tables and Pivot Charts.
* Build an interactive Business Intelligence Dashboard.
* Identify key business trends and performance metrics.

## 🛠️ Tools Used

* Dashboard Design
